# Knowledge System V1

## Purpose

This repository is a specialized CRO application. It should not become the master knowledge store for all DTC research.

The long-term system has three layers:

1. **Shared DTC knowledge core**
   - Podcasts
   - Books
   - Newsletters
   - Internal research
   - Cases
   - Experiments
   - Decisions

2. **Specialized applications**
   - Operators Vault / DTC wiki
   - Billion Dollar CRO Analyzer
